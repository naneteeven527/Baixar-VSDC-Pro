## 📋 Troubleshooting Common Issues

If you experience problems, check these items first.

*   **Slow Performance:** Close other demanding applications like browsers or games while you edit.
*   **Import Errors:** Verify the video format. Most standard formats like MP4, AVI, and MOV work well. If a file fails to import, ensure the file is complete and not corrupted.
*   **Missing Audio:** Check the track settings in the timeline. Ensure the track is not muted.
*   **Export Failures:** Check the remaining space on your hard drive. Large files require significant disk space during the rendering process.
